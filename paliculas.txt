Evidencia de aprendizaje: API REST - NodeJs

Objetivo:
Desarrollar una API REST utilizando JavaScript del lado del servidor con NodeJs.

Instrucciones:

Aspectos a tener en cuenta para construir la API REST:
Diseñar la base de datos.
Construir los distintos métodos que realizaran las operaciones requeridas de la API (POST, PUT, DELETE, GET, etc.).
El funcionamiento de la API se realizará utilizando postman o algún otro cliente que el estudiante considere.
Para esta actividad el estudiante se apoyará en el material dado por el tutor en los encuentros sincrónicos y anuncios a través de la plataforma de Canvas. Por lo tanto, se recomienda asistir a los encuentros o mirar las grabaciones de las sesiones y estar activos en la plataforma de Canvas.
